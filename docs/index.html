Syncd - 自动化部署工具

Syncd是一款开源的代码部署工具，它具有简单、高效、易用等特点，可以提高团队的工作效率。

开始使用 Github Gitee

特性

  • Go语言开发，编译简单、运行高效
  • Web界面访问，交互友好
  • 权限模型灵活自由
  • 自定义构建支持
  • Git仓库支持
  • 部署Hook支持，可扩展性强
  • 完善的上线工作流
  • 邮件通知机制 
$ curl https://syncd.cc/install.sh | bash
$ cd ./syncd-deploy
$ mysql --default-character-set=utf8mb4 < ./resource/sql/syncd_{版本号}.sql
$ GIN_MODE=release ./bin/syncd -c ./etc/syncd.ini
